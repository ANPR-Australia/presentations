Automatic Number Plate Recognition (ANPR)

A 6 week experiment using Deep Learning

DCS Transformation Team - Emerging Technology

Why ANPR?

Get a computer to perform a tedious, repetitive and expensive task.

How?

Deep Learning

Deep Learning

Why Deep Learning?

  • Deal with the diversity of Australian plates - at least 15,000 unique designs
  • Can learn to recognise plates better as it's fed more labeled data.

Results

  • We can identify a number plate in an image with >95% accuracy
  • We can read the text on it with 70% accuracy

Results

  • As a proof of concept, the system was trained on a pre-existing dataset of foreign plates
  • Using Australian data, every time an infringement is processed - we can use the labeled data to further train the system

Training and Augmentation

  • With training, the system will get better at recognising Australian plates
  • We can augment other data sources, like Drives
  • We can incorporate new technology as it appears

Improved Experience for Staff

  • Have higher throughput
  • Fewer errors
  • Less fatigue, more interesting work
