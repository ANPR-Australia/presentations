Automatic Number Plate Recognition

Deep Learning with YOLO

Why ANPR

Get a computer to perform a tedious, error prone, repetitive and expensive task.

How?

Deep Learning

Why Deep Learning?

  • Deal with the diversity of Australian plates - at least 15,000 unique designs
  • Can learn to recognise plates better as it's fed more labeled data.

Results

  • We can identify a number plate in an image with >95% accuracy
  • We can read the text on it with 70% accuracy
  • Overall >65% of number plates read correctly

They'll only get better!

  • Training!
  • Augmenting data sources
  • Incorporating new technology as it appears

This is just the beginning!

Improved Experience for Staff

  • More throughput
  • Fewer errors
  • Less fatigue, more interesting work

What's Next?

We can use our Deep Learning capability to tackle other problems. For example, common exceptions.

Cross Government Utility

  • Useful for:
